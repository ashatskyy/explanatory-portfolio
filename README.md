# English Explanatory Dictionary Online

## Description
Pixel-perfect implementation based on a Figma design file for an English Explanatory Dictionary website

## Features
- Supports: light and dark mode
- Allows: users to switch between three different fonts
- Responsive design: works seamlessly on desktop, tablet, and mobile devices
- Renders correctly: across all modern browsers
- Fully integrates: with all social media platforms, including banners with images
- Each page: accessible via its own unique URL

## Technologies
- HTML
- SCSS
- React
